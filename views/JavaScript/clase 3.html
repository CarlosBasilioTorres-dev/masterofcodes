Hola Mundo en JavaScript

Publicado el 5 de mayo de 2025

Una vez entendida la teoría detrás de como funciona Java. Comenzaremos con la creación de nuestro primer código en este lenguaje. 

public class HolaMundo{
    public static void main(String []argv){
        System.out.println("Hola mundo");
    }
}

Ahora que lo hemos escrito, es momento de correrlo en tu ide:

  • Inicia un nuevo proyecto dentro de tu IDE (acorde a la que utilices)
  • Crea un archivo de nombre HolaMundo.java
  • Escribe este código tal cual se muestra.
  • Córrelo (nuevamente, acorde a como se realice en tu IDE este paso)

Si a este punto ha salido todo bien, deberías de ver la siguiente respuesta: 

Hola mundo

Sencillo ¿no te parece?. Pues aunque en principio lo es, hay mucho que estudiar en este pequeño código, nuevamente intentaremos hacerlo fácil, sencillo y para toda la familia.

Creación de una clase 

public class HolaMundo{
            
}

Las "clases" en Java son el elemento principal de creación. En esencia son "contenedores" que almacenan código relacionado entre si.

Nos indica que es una clase pública, lo cual en resumidas cuentas se interpreta como el hecho de poder ser visualizada por cualquier otra clase que lo necesite.

Se que de momento esto no nos dice mucho, veras que conforme pasen las clases, todo será mas claro.

El método principal

public class HolaMundo{
    public static void main(String []argv){
                
    }
}

El siguiente apartado es conocido como el método principal, es decir, el método que Java buscara de primera mano y de donde comenzará a ejecutar nuestro código, dicho de otra manera, es el método que Java ejecuta primero al iniciar una aplicación.

Método println

public class HolaMundo{
    public static void main(String []argv){
        System.out.println("Hola mundo");
    }
}

Por último pero no menos importante, tenemos un método llamado println que no es mas que el método mas sencillo de escribir un mensaje en consola. Lo peculiar de este método es que en al terminar de escribir una linea, en automático da un salto de linea (el equivalente a presionar enter en un editor de texto como word).
